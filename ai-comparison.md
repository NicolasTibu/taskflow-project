# Comparativa de Modelos de IA

## Introduccion 
En este documento se realizará un análisis comparativo entre los diferentes Modelos de Lenguaje (LLMs) utilizados durante el proyecto (ej. GPT-4o y Claude 3.5 Sonne). El objetivo es identificar las fortalezas y debilidades de cada uno en tareas específicas de programación y razonamiento.

## Modelos Evaluados
* **Modelo A:** Chat GTP
* **Modelo B:** Claude 

## Tabla Comparativa 
| Característica | Modelo A | Modelo B |
| :--- | :--- | :--- |
| Velocidad | Alta (Respuesta instantánea) | Media (Procesamiento más pausado) |
| Calidad de Código | Funcional y simplificado | Estructurado y con mejores prácticas |
| Seguimiento de Instrucciones | Muy bueno (Incluye tabla resumen) | Excelente (Uso de emojis y formato limpio)  |
| Profundidad Técnica | Nivel básico/Intermedio | Nivel intermedio/avanzado

# Análisis de Contenido

**Modelo A:** ChatGPT
* **Fortalezas:** Muy didáctico para principiantes. El ejemplo del DOM es fundamental para entender para qué sirve JS en la web. La tabla resumen al final es un gran acierto para el repaso rápido.

* **Debilidades:** La explicación del Hoisting es correcta pero menos detallada sobre la "Temporal Dead Zone" de let y const.

**Modelo B:** Claude

* **Fortalezas:** Explicación magistral del Event Loop, un concepto complejo que explicó con un flujo lógico muy claro. Sus ejemplos de Closures incluyen comentarios que explican el "porqué" sobrevive la variable.

* **Debilidades:** No incluyó una tabla resumen (aunque su formato de lista es muy legible).

# Conclusión

Para entender cómo funciona el lenguaje por dentro (arquitectura), Claude ofrece una profundidad superior. Para conceptos rápidos y aplicaciones prácticas directas (como manipular la web), ChatGPT es más directo y efectivo.
