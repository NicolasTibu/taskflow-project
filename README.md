# 📝 Taskflow - Aplicación Dinámica de Tareas

Taskflow es una aplicación web interactiva desarrollada con HTML, CSS (Tema Oscuro Moderno) y Vanilla JavaScript. Permite a los usuarios gestionar sus tareas diarias con persistencia de datos local.

## ✨ Características (Features)
- **Añadir tareas:** Interfaz rápida e intuitiva.
- **Borrar tareas:** Eliminación dinámica del DOM.
- **Persistencia de datos:** Uso de `localStorage` para no perder la información al refrescar la página.
- **Búsqueda en tiempo real (Bonus):** Filtro interactivo que oculta tareas que no coinciden con la búsqueda.
- **Diseño Moderno:** Interfaz responsiva con *Dark Mode* y animaciones suaves.

## 🛠️ Tecnologías utilizadas
- HTML5
- CSS3 (Flexbox, animaciones)
- JavaScript (ES6+, DOM Manipulation, LocalStorage)
