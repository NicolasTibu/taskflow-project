# Flujo de Trabajo con Cursor

## Flujo de Trabajo con Cursor

## Introducción
Este documento detalla la metodología y las mejores prácticas adoptadas al utilizar el IDE Cursor. Se busca documentar cómo se integran las funciones de IA (Composer, Chat, Tab) en el ciclo de vida del desarrollo.

## Configuración del Entorno 
* Extensiones clave. 
* Reglas del sistema (.curorrules).

## Estrategias de Uso 
* Uso de `Ctrl+K` para refactorización rápida.
* Gestión de archivos mediante el Composer (`Ctrl+I`).

## Atajos de teclados que he usado con más frequencia 

| Comando | Acción | Propósito |
| :--- | :--- | :--- |
| Ctrl + K | Edición Inline | Modificar o generar bloques de código en el archivo actual |
| Ctrl + L | Chat Contextual | Abrir el panel lateral para preguntar sobre el código seleccionado |
| Ctrl + I | Composer | Generar cambios complejos que afectan a múltiples archivos a la vez  |
| Ctrl + Enter | Aceptar sugerencia | Aplicar los cambios propuestos por la IA directamente al archivo |
